# Real Estate Management System

A web-based **Real Estate Management System** developed using **Core PHP, MySQL, HTML, CSS, and JavaScript**. This project allows users to browse property listings, search and filter properties, and contact agents, while administrators can manage properties, users, and inquiries through a secure dashboard.
